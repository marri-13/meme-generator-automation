# 🤖 AI Meme Generator Automation (Make.com)

This project is an automation workflow built using Make.com.

## 🔄 Workflow Overview
1. Reads emails from Gmail
2. Extracts links using text parser
3. Fetches article content via HTTP
4. Uses Gemini AI to generate meme text
5. Creates meme using Imgflip API
6. Sends generated meme via email

## 🛠 Tools Used
- Make.com
- Gmail API
- Gemini AI (Google)
- Imgflip API
- HTTP modules

## 📁 Files
- scenario.json → Make.com blueprint

## 🚀 How to Use
1. Import `scenario.json` into Make.com
2. Connect your Gmail & Gemini accounts
3. Run the scenario

## 📌 Note
This is an automation workflow, not a traditional code-based project.
